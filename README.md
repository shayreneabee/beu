# BEU 🌍

A modern web application designed to explore culture, travel, and experiences through a structured compass-style interface.

## 🛠 Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## 🚀 Features
- Component-based UI
- Compass navigation system
- Dynamic sections (Cuisine, Places, Events)

## ▶️ Run Locally
npm install
npm run dev
